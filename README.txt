# Coach Bertin V50.18 — fichiers data initiaux optionnels

À copier dans `data/` seulement si le fichier manque.

Rôles :
- `resultats.json` : journal brut. Ne jamais réécrire l'histoire.
- `athlete_state.json` : force actuelle estimée par mouvement. Pas de XP, pas de level, pas de jeu vidéo.
- `cycle_state.json` : cycle actif, semaine, journée, jours complétés.
- `charges.js` : charges de base / équipement / préférences. Ce n'est pas ta force actuelle.

Important :
- Ne jamais écraser un fichier `data/` qui contient déjà tes vraies données.
- Les programmes dans `programs/` donnent le plan.
- Les fichiers `data/` sont tes données durables.
