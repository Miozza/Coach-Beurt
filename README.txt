# Coach Bertin — fichiers data initiaux

À mettre dans le dossier `data/` du repo GitHub.

Fichiers :
- `resultats.json` : historique brut des séances.
- `athlete_state.json` : niveau réel / recalibrage par mouvement.
- `cycle_state.json` : cycle actif, semaine, journée, jours complétés.
- `charges.js` : charges de base / équipement. Ce n'est pas ta capacité réelle.

Important :
- Si un de ces fichiers existe déjà et contient des données, ne l'écrase pas.
- Si ton dossier `data/` est vide, tu peux ajouter ces fichiers.
